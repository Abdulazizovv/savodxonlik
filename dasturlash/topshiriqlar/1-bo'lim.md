# Amaliy topshiriqlar
umumiy 1-bo'lim bo'yicha o'tilganlar bo'yicha topshiriqlar

## 1-topshiriq
foydalanuvchi 1 dan 12 gacha(12 ham) son kiritsin. Shu songa mos oy nomi ekranga chiqsin. Masalan:
```
Oy tartib raqamini kiriting: 1
OY: Yanvar
```


## 2-topshiriq
Foydalanuvchidan bo'yini (metrda) va vaznini (kg da) so'rang. BMI (Body Mass Index) hisoblang va natijani baholang:
- BMI < 18.5: "Siz ozg'insiz"

- 18.5 ≤ BMI < 25: "Sizning vazningiz normal"

- 25 ≤ BMI < 30: "Siz ortiqcha vaznga egasiz"

- BMI ≥ 30: "Siz semizlik darajasidasiz"

```
Bo'yingizni (metrda) kiriting: 1.75
Vazningizni (kg da) kiriting: 70
Sizning BMI: 22.86
Sizning vazningiz normal
```

*Bu topshiriqda internetdan foydalangan holda BMI hisoblash formulasini toping!*

## 3-topshiriq
Parol generator dasturi. Foydalanuvchidan parol uzunligi so'ralsin va shu uzunlikdagi tasodifiy parol generatsiya qilinsin
```
Parol uzunligini kiriting: 12
Generatsiya qilingan parol: aB3$kL9@mN2!
```

## 4-topshiriq
Studentlar haqida ma'lumot saqlovchi lug'at yarating. Har bir student uchun: ismi, yoshi, yo'nalishi va baholari (3 ta fan) saqlansin.

- Barcha studentlarni chiqaring

- Yoshi 20 dan katta bo'lgan studentlarni toping

- O'rtacha bahosi eng yuqori bo'lgan studentni aniqlang

```
students = {
    'Ali': {'age': 19, 'major': 'CS', 'grades': [85, 90, 88]},
    'Vali': {'age': 21, 'major': 'Math', 'grades': [92, 88, 94]},
    'Hasan': {'age': 20, 'major': 'Physics', 'grades': [78, 85, 80]}
}
```


## Qo'shimcha masalalar (murakkab)
A. Telefon kitobi
Telefon kitobi dasturini yozing. Quyidagi funksiyalarni amalga oshirsin:

Kontakt qo'shish (ism, telefon, email)

Kontaktni qidirish

Barcha kontaktlarni ko'rsatish

Kontaktni o'chirish

Faylga saqlash / fayldan yuklash

B. Online do'kon tizimi
Oddiy online do'kon tizimi. Quyidagilarni qo'llab-quvvatlashi kerak:

Mahsulotlar katalogi (nom, narx, miqdori)

Savatga qo'shish / savatdan olib tashlash

Xaridni amalga oshirish

Buyurtmalar tarixi

Foydalanuvchi hisobi (register/login)
